# JC Compose Demo

A minimal Android demo app using Kotlin and Jetpack Compose.

## Prerequisites
- Android Studio Koala+ (or latest)
- Android Gradle Plugin 8.5+
- JDK 17

## Getting Started
1. Open this folder in Android Studio: `File` → `Open` → select the project root.
2. Let Gradle sync finish (it will download dependencies).
3. Select an Android device/emulator.
4. Run the `app` configuration.

## Tech Stack
- Kotlin
- Jetpack Compose (Material 3)
- AndroidX Activity Compose

## Notes
- Min SDK 24, Target/Compile SDK 35.
- Compose BOM is used to manage Compose versions.
